## Overview

At Sahu Technologies, I worked as a Software Developer Intern focusing on cloud-native infrastructure and deployment automation. My responsibilities centered around optimizing cloud resources, setting up continuous delivery workflows, and improving system observability. This role provided deep exposure to DevOps tools and AWS services in a production environment.

## Responsibilities

- Deployed and managed scalable services using AWS EC2, S3, RDS, and CloudWatch.
- Reduced infrastructure costs by 35% through efficient provisioning and resource tuning.
- Built and maintained CI/CD pipelines with Jenkins, Docker, and Terraform, improving deployment speed by 40%.
- Configured logging and system health alerts with AWS CloudWatch to monitor uptime and detect failures early.
- Collaborated with senior developers on infrastructure-as-code and release automation.
